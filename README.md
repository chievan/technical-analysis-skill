# Technical Analysis Skill

This repository is used to store, manage, and version control the custom **Technical Analysis Pro** agent skills and indicator computation script rules used in our technical analysis platform.

## Contents
- `SKILL.md`: The system prompt and behavior definitions for the AI Technical Analyst.
- `scripts/`: Python K-line calculation engine scripts (e.g., `ta_engine.py`, indicators).
- `references/`: Knowledge references (e.g., Japanese candlestick patterns, support/resistance models).
